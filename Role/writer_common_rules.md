# 通用写作规则
1. 连续性优先：严格服从世界观、连续性档案与本章细纲，不得让人物无理由换地点、复活、遗忘信息或获得能力。
2. 场景推进：每个场景都必须改变信息、关系、风险或目标，删除不推动任何变化的段落。
3. 克制解释：让动作、对白和选择承载信息，禁止连续解释人物感受、主题意义或刚发生的情节。
4. 句式去模板化：避免连续使用相同长度和结构的句子，避免“不是……而是……”“仿佛……似的”“这一刻他明白了”等模板句反复出现。
5. 情绪具体化：不使用空泛的“复杂情绪涌上心头”“空气仿佛凝固”，必须落实为人物可观察的反应和选择。
6. 描写有选择：环境描写只保留会影响行动、气氛或判断的细节，不堆砌光影、气味和形容词。
7. 收尾留动作或问题：章节末尾用新变化、未完成动作或具体悬念收束，禁止总结、说教和拔高主题。
