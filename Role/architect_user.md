# 任务描述（Task）
用户的初始灵感是：【{user_idea}】

【本次已确认结构化素材】：
{materials}

【篇幅要求】：{chapter_requirement}
请构建世界观，并规划详细剧情细纲。

强制要求：每一项确认素材都必须有机融入世界观、人物或主线事件，不得只在设定中提到一次便丢弃。
强制要求：素材必须服从主套路；世界舞台、人物、冲突与装置只能补充主线，不得替代主套路核心驱动力。
强制要求：每一章细纲去除空白后必须不少于200字，且必须包含可直接指导写手落笔的具体事件、人物行动、冲突推进与结尾钩子。chapter_outlines 的每章值必须是纯字符串，且第一个非空字符必须是【开场状态】，不得输出嵌套对象、字段字典、章节标题或小标题。章节细纲只能使用固定中括号标签：非最终章使用【开场状态】【核心冲突】【关键行动】【人物关系变化】【重要信息或伏笔】【结尾结果】【下一章钩子】；最终章使用【开场状态】【核心冲突】【关键行动】【人物关系变化】【重要信息或伏笔】【全书结局】。
强制要求：每章只安排一个主冲突、一个关键转折和一个关系变化；关键事件必须能场景化，禁止用“几天后/很快/最终”把核心对抗、摊牌、说服、暴露、背叛或反杀摘要跳过。
强制要求：最终章必须解决主冲突、交代主要人物命运和关系落点，并给出明确结束信号；禁止开启新任务、新反派或续集主线。
